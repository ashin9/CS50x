## Readability

### 问题

给一段文字, 评判等级

### 背景

单词长度, 句子长度, 衡量高阅读水平特征

Coleman-Liau 指数公式 : `index = 0.0588 * L - 0.296 * S - 15.8`

- `L` 每 100 个单词的平均字符数
  - 空格分隔为单词，空格数+1 = 单词数

- `S` 每 100 个单词的平均句子数
  - `. ? !` 标记句子结束，标点数 = 句子数

index 四舍五入，指数越高级别越高

### 特殊处理

- Before Grade 1

- Grade 16+



## caesar

凯撒加密



接受参数, 判断是否符合格式

- 参数为字符串, 需要遍历检查每个字母是否为数字, 来检查是否为数字

```c
bool only_digit(string arg)
{
    for (int i = 0, len = strlen(arg); i < len; i++)
    {
        if (!isdigit(arg[i]))
        {
            return false;
        }
    }
    return true;
}
```

加密

- 分大写,小写,字符情况, 分别加减模运算即可
